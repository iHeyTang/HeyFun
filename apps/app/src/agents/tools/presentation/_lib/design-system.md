# PPT 设计系统优化方案

## 一、智能布局系统

### 1.1 布局策略选择

根据内容类型和数量，自动选择最佳布局：

```typescript
// 布局选择规则
- 只有标题 → 'title' 布局（居中大标题）
- 标题 + 1-3个要点 → 'title-content' 布局（标题在上，内容在下）
- 标题 + 4+个要点 → 'two-column' 布局（两列显示）
- 标题 + 图片 → 'image-left' 或 'image-right'（根据内容选择）
- 只有图片 → 'image-full' 布局（全屏图片，标题叠加）
- 章节分隔 → 'section' 布局（大标题居中）
```

### 1.2 图片布局优化

**问题**：当前图片太小，位置固定

**解决方案**：

- **全屏背景模式**：图片作为背景，文字叠加（适合视觉冲击）
- **左右分栏模式**：图片占40-50%，文字占50-60%（适合说明性内容）
- **上下分栏模式**：图片在上或下，占40-50%（适合流程说明）
- **卡片模式**：图片作为卡片，文字环绕（适合产品展示）

### 1.3 响应式布局

根据内容量动态调整：

- 内容少 → 增大字体，增加留白
- 内容多 → 减小字体，优化间距
- 图片多 → 使用网格布局

## 二、配色系统增强

### 2.1 语义化配色

根据内容主题自动选择配色：

```typescript
// 配色规则
- 科技/技术 → 蓝色系（#2563eb, #3b82f6）
- 商业/金融 → 深蓝/灰色系（#1e3a8a, #64748b）
- 创意/艺术 → 渐变色（紫色到粉色）
- 教育/学习 → 绿色系（#059669, #10b981）
- 医疗/健康 → 蓝绿色系（#0891b2, #14b8a6）
- 环保/自然 → 绿色系（#16a34a, #22c55e）
```

### 2.2 对比度优化

- 确保文字和背景对比度 ≥ 4.5:1（WCAG标准）
- 自动调整文字颜色以确保可读性

### 2.3 渐变背景

- 使用渐变背景增加视觉层次
- 根据主题选择渐变方向（水平/垂直/对角）

## 三、设计规则引擎

### 3.1 视觉层次

- **标题层级**：H1(48px) > H2(32px) > H3(24px) > Body(20px)
- **间距系统**：8px 基础单位，使用 8/16/24/32/40/48px
- **留白规则**：内容区域不超过80%，保持20%留白

### 3.2 图片处理

- **尺寸规则**：
  - 全屏背景：100% × 100%
  - 分栏图片：40-50% × 60-80%
  - 卡片图片：30-40% × 40-60%
- **圆角**：8-12px（现代感）
- **阴影**：0 4px 12px rgba(0,0,0,0.15)（层次感）

### 3.3 文字排版

- **行高**：标题 1.2，正文 1.6-1.8
- **字间距**：标题 -0.5px，正文 0px
- **对齐**：标题居中或左对齐，正文左对齐

## 四、AI 辅助设计

### 4.1 内容分析

AI 应该分析：

- 内容类型（技术/商业/创意等）
- 内容密度（文字量）
- 是否需要配图
- 配图类型（图标/照片/图表）

### 4.2 设计建议

AI 应该提供：

- 布局建议（根据内容选择）
- 配色建议（根据主题选择）
- 配图建议（搜索关键词建议）

## 五、实现优先级

### Phase 1: 布局优化（立即）

1. 修复图片布局问题（增大尺寸，智能位置）
2. 实现左右分栏布局
3. 实现全屏图片布局

### Phase 2: 配色增强（短期）

1. 实现语义化配色
2. 添加渐变背景支持
3. 优化对比度

### Phase 3: 设计规则（中期）

1. 建立设计规则库
2. 实现视觉层次系统
3. 优化间距和留白

### Phase 4: AI 增强（长期）

1. 内容分析系统
2. 自动设计建议
3. 智能配图选择
